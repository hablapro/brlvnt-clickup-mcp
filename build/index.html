About

This is a ClickUp MCP (Model Context Protocol) Server that enables AI assistants to interact with ClickUp workspaces. It provides tools for task management, project organization, and team collaboration through the MCP protocol.

Features

  • Task creation, updating, and management
  • Workspace and project hierarchy navigation
  • Time tracking and logging
  • Document and comment management
  • Real-time event streaming
  • Webhook integration
